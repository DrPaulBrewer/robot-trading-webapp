Tips:

  • Do not use trash to clear the screen. Simply select >Run to run again and the screen will clear automatically. Refreshing the browser window also clears the previous run.
  • Trash is used only to discard useless scenario parameter settings, such as typos, incorrect settings, or obsolete settings no longer needed.
  • Do not trash scenarios that other people might be looking for later.
  • If you trash some parameters and later want those restored, look in the trash tab
  • Clicking the Keep button is safe and does nothing beyond dismissing this message
